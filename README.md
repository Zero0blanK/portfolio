# Personal Portfolio

A modern, layered portfolio built with Next.js, React, Framer Motion, and Tailwind CSS. It combines polished motion, strong typography hierarchy, and reusable surface patterns for a distinctive but performant UX.

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS v4 with custom CSS variables
- **Animation**: Framer Motion for smooth, performant transitions
- **Intersection Detection**: react-intersection-observer for scroll triggers
- **TypeScript**: Type-safe component development
